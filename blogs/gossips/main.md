# 2021年：光阴里的便利贴
> 光阴里的便利贴指的是以前创作的随笔。这些随笔都是从以前不经意留下的文字中收集而来的，它们有的原本留在已经弃坑的社交软件账号中，有的散落在旧手机的备忘录里；有的记在英语六级考试发的草稿纸上，有的赫然出现在高中二模试卷的答题卡背面...当时怎么会写下这些文字连作者本人都记不清了，但是就像毕业季整理东西时不经意间从书本中抖出了一张便利贴，便会拿起来煞有其事地仔细阅读，加载那段似梦的回忆。
